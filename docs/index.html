芳香族マスター

はじめに
高校の化学で学習する「芳香族化合物」。ベンゼンを初めに、化合物名、反応式、付加反応・置換反応、スルホン化などの反応名と覚える項目が多く、悩む学生さん達が多いと思います。このノード・リンク図は、そんな芳香族をまとめて覚えてしまおう！という名目のもと、開発しました。多くの方々にご覧いただければ幸いです。
